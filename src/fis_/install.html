Fis 可视化工具

您的系统中有未安装的软件 (请按照顺序安装) ---安装完成之后请重启软件

安装Node

  • 下载软件

  • 运行安装包全部按照默认的安装

安装Java

  • 下载软件

  • 运行安装包全部按照默认的安装

安装PHP

  • 下载软件

  • 请下载Thread Safe 的 zip 版本
  • 下载完成后解压到某个盘中 (推荐 C盘根目录下)
  • 配置解压路径的PHP的环境变量 （我的电脑 > 右键 > 属性 > 高级系统设置 > 高级 > 环境变量）
  • 在系统变量中找到Path变量(如果没有的话请新建一个),添加解压路径，与前面的路径以分号隔开

安装FIS (请先确保安装了NodeJs)

  • 打开命令行 ( win + R 然后输入cmd )
  • 输入 npm i -g fis 等待安装完成
  • 安装完成后可使用 fis -v 检验安装情况，如果出现版本信息则表示安装成功
